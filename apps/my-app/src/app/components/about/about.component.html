3Dice


Hello! 3Dice is a client-side web programming platform created by Nick Thanh Phong Nguyen - 2223623


The goal of 3Dice is to create a platform for everyone who has an interest in 3D printing!

Whether you're a hobbyist, designer, or maker, our mission is to make 3D printing as accessible as possible.


What Can You Do on 3Dice?
  • Explore 3D models: Browse a wide range of 3D models uploaded by the community.
  • Download 3D models: Save and use models for your own 3D printing projects.
  • Upload 3D models: Share your creations with the 3Dice community.
  • Share 3D models: Easily distribute your designs through links or embeds.
  • Rate 3D models: Provide feedback and ratings to help others discover the best designs.

Functional requirements (FR)
ID User Story Acceptance Criteria Priority (MoSCoW)
1 As a user, I want to be able to upload 3D models so that I can share my designs with others.
  • Users can upload a model file (e.g., .STL).
  • A title, description, and category must be filled in.
  • Only logged-in users can upload.
 Must
2 As a user, I want to be able to download 3D models so that I can use them for my own projects.
  • Download option available on the detail page of each model.
  • Only registered users can download.
 Must
3 As a user, I want to be able to rate 3D models so that I can see their quality and popularity.
  • Users can give a star rating (1-5).
  • An average score is displayed on the model page.
  • Only logged-in users can rate.
 Should
4 As a user, I want to be able to view a list of available 3D models so that I can get inspiration.
  • Models are clickable for more details.
 Must
5 As a user, I want to be able to manage my own uploaded models so that I can make changes.
  • Users can edit or delete their own models.
  • Changes require login.
  • Other users do not have access to edit/delete.
 Must
