3Dice


Hallo! 3Dice is een client-side web programming website van Nick Thanh Phong Nguyen - 2223623


Het doel van 3Dice is om een platform te creeëren voor iedereen, die interesse heeft in 3d printen!

Of je nou een hobbyis, ontwerper of maker bent, ons doel is om 3D-printen zo toegankelijk mogelijk te maken.


Wat kan je nou op 3Dice?
  • 3D-modellen bekijken
  • 3D-modellen downloaden
  • 3D-modellen uploaden
  • 3D-modellen delen
  • 3D-modellen beoordelen

Schrijf je nu in!

Functionele requirements (FR)
ID User Story Acceptatiecriteria Prioriteit (MoSCoW)
1 Als gebruiker wil ik 3D-modellen kunnen uploaden, zodanig dat ik mijn ontwerpen kan delen met anderen.
  • Gebruikers kunnen een modelbestand (bijv. .STL) uploaden.
  • Een titel, beschrijving en categorie moeten ingevuld worden.
  • Alleen ingelogde gebruikers mogen uploaden.
 Must
2 Als gebruiker wil ik 3D-modellen kunnen downloaden, zodanig dat ik deze kan gebruiken voor mijn eigen projecten.
  • Downloadoptie beschikbaar op de detailpagina van elk model.
  • Alleen geregistreerde gebruikers kunnen downloaden.
 Must
3 Als gebruiker wil ik 3D-modellen kunnen beoordelen, zodanig dat ik de kwaliteit en populariteit kan zien.
  • Gebruikers kunnen een sterrenscore (1-5) geven.
  • Een gemiddelde score wordt weergegeven op de modelpagina.
  • Alleen ingelogde gebruikers mogen beoordelen.
 Should
4 Als gebruiker wil ik een lijst van beschikbare 3D-modellen kunnen bekijken, zodanig dat ik inspiratie kan opdoen.
  • Modellen zijn klikbaar voor meer details.
 Must
5 Als gebruiker wil ik mijn eigen geüploade modellen kunnen beheren, zodanig dat ik wijzigingen kan doorvoeren.
  • Gebruikers kunnen hun eigen modellen bewerken of verwijderen.
  • Wijzigingen vereisen inloggen.
  • Andere gebruikers hebben geen toegang tot bewerken/verwijderen.
 Must
